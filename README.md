# PROVEXA — Sitio web

Sitio web estático para la empresa PROVEXA (Proveedora por Excelencia). Diseño elegante y profesional, paleta basada en el logo proporcionado.

Cómo previsualizar

1. Coloca el logotipo en `assets/logo.png` (reemplaza el placeholder si lo hay).
2. Abre `index.html` en tu navegador o sirve la carpeta con un servidor estático, por ejemplo:

```bash
# desde la carpeta del proyecto
python -m http.server 8000
# o con Node.js
npx serve .
```

Recomendaciones de despliegue

- GitHub Pages (push a un repo y activar Pages desde la rama `main`).
- Netlify o Vercel para despliegue automático desde Git.

Notas

- Los formularios son de ejemplo y no envían datos a un backend.
- Colores principales: Azul marino `#082A66`, Gris plata `#8D8D8D`, Blanco `#FFFFFF`.
